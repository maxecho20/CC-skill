---

这个工作流会自动对指定主题进行深度研究，并生成适配不同平台的文稿和视觉图。

### 步骤

1. **深度研究**：
   - 使用 `search_web` 搜索主题最新的资讯、技术细节和新闻。
   - 分析主题的核心卖点、发布日期、关键特性及价格/性能指标。

2. **内容创作**：
   - **长文 (WeChat)**：生成深度分析文章，包含引言、分点阐述和总结。
   - **X (Twitter)**：生成简洁有力、带标签的亮点总结文案。
   - **小红书 (XHS)**：生成生动活泼、带表情包、具有“种草”感的笔记文案。

3. **视觉生成**：
   - 使用 `generate_image` 为主题生成至少 2 张高质量的配图（海报、UI 或相关意向图）。

4. **自动化归档**：
   // turbo
   - 执行以下命令创建归档文件夹（命名格式：YYMMDD_主题）：
     ```bash
     DATE=$(date +%y%m%d)
     FOLDER_PATH="/Users/maxecho_1/编程项目0408/claude skills/wengaoxiezuo/${DATE}_${TOPIC}"
     mkdir -p "$FOLDER_PATH"
     ```
   - 将生成的文稿保存为 `wechat.md`, `x.md`, `xiaohongshu.md`。
   - 将生成的图片保存到该文件夹下。

### 使用方法

在对话中输入：`/XZ [主题名称]`

### 注意事项
- 如果 `/Users/maxecho/Documents` 权限受限，默认归档至项目内的 `outputs/` 目录。
- 确保联网以获取最新资讯。
